📖 Overview

This software helps players see information during Call of Duty matches. It provides data overlays and mechanical assistance to help you track game objects. The tool includes features like ESP, Aimbot, Wallhack, No Recoil, and a Triggerbot. These modules help users manage game information and weapon handling under various conditions.

⚙️ Features

  • ESP: Displays outlines around players and objects.
  • Aimbot: Adjusts camera movement toward targets.
  • Wallhack: Visualizes markers through solid game geometry.
  • No Recoil: Stabilizes weapon kick patterns automatically.
  • Triggerbot: Fires automatically when your crosshair detects an enemy.

  • Graphics: DirectX 12 compatible card.

🛡️ Usage Tips

This tool functions as an overlay. Ensure you run your game in "Borderless Windowed" mode for the best results. If the overlays do not appear, restart the application after the game starts. Use the on-screen menu to toggle specific features during your session. Keep the application updated to the latest version to maintain compatibility with game patches.

If the application fails to open, verify that your antivirus or Windows Defender is not blocking the process. Some security software detects overlay tools as potential threats. Add an exclusion folder if needed. Ensure your graphics drivers are current. If you experience performance drops, lower your frame rate settings inside the game menu.
